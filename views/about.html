Documentação API
Quero entender
Acesso à API
Cadastro de uma biblioteca
Ediçao de uma biblioteca
Apagando uma biblioteca
Esta rota retorna todos as libs cadastradas
Chamando a API com o metodo POST voce conseguira adicionar uma nova biblioteca, mas deve passas os parametros a baixo
Chamando a API com o metodo PUT voce conseguira editar info de biblioteca, passando o id da lib na URL e APENAS o parametro que deseja editar no bady da req
Chamando a API com o metodo DELETE voce conseguira APAGAR uma biblioteca, passando o id da lib na URL
id: Gerado automaticamente, nao deve ser passado na req
name: Deve ser informado na req
description: Deve ser informado na req
tecnologia: deve ser informado na req
logo: deve ser informado na req
Basico
Props de uma biblioteca
Todo acesso retornará uma mensagem de erro ou sucesso (message).
